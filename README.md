# kotlin-cli-seg-sieve

## Description
Given a range, print all prime numbers.

## Tech stack
- kotlin

## Docker stack
- gradle:4.2.1-jdk8-alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
